# 📱 מדריך פקודות מלא - סוכן וואטסאפ AI

## 🎯 איך להשתמש בסוכן

כל הפקודות נשלחות **לעצמך** בצ'אט עצמי (self-chat) בוואטסאפ.

---

## 🤖 פקודות ג'מיני AI

### שאלות ופניות כלליות
```
ג'מיני, מה מזג האוויר היום בתל אביב?
ג'מיני, תסכם לי את החדשות האחרונות
ג'מיני, איך מכינים פיצה בבית?
```

### שליחת הודעות לאנשי קשר
```
ג'מיני, תגיד לאשתי שאני אגיע מאוחר
ג'מיני, תשלח לאבא שאני בסדר
ג'מיני, תודיע לחברים שהמסיבה בוטלה
```

**הערה:** הסוכן יחפש אוטומטית את איש הקשר לפי השם שכתבת.

## 🛒 ניהול רשימות קניות

### יצירת רשימה חדשה או הוספה לקיימת
```
רשימת קניות: חלב, לחם, ביצים, עגבניות
```

### יצירת רשימה עם שם ספציפי
```
רשימת קניות [שבועי]: חלב, לחם, ביצים
רשימת קניות [חג הפסח]: מצות, יין, עוף
רשימת קניות [סופר]: שמן, סוכר, קמח
```

### מחיקת פריט ספציפי מרשימה
```
מחק מרשימת קניות: חלב
מחק מרשימת קניות [שבועי]: ביצים
```

### שיתוף רשימה עם איש קשר
```
שתף רשימת קניות עם [שם איש הקשר]
שתף רשימת קניות [שם הרשימה] עם [שם איש הקשר]
```
**דוגמאות:**
- `שתף רשימת קניות עם אשתי`
- `שתף רשימת קניות [שבועי] עם חבר`

### מחיקת רשימה שלמה
```
מחק רשימת קניות
מחק רשימת קניות [שבועי]
```

### מחיקת כל רשימות הקניות
```
מחק את כל רשימות הקניות
```

## 🎯 דוגמאות שימוש מלאות

### תרחיש 1: יום רכישות
```
1. רשימת קניות: חלב, לחם, ביצים
2. רשימת קניות: שמנת, גבינה, ירקות
3. שתף רשימת קניות עם אשתי
4. מחק מרשימת קניות: שמנת (אם לא צריך יותר)
```

### תרחיש 2: פגישה עסקית
```
1. הודעה: "מחר ב-14:00 יש פגישה עם הלקוח"
2. הסוכן שואל אם להוסיף ליומן
3. ענה: "כן"
```

### תרחיש 3: איש קשר חדש
```
1. מישהו לא מוכר שולח: "היי, זה דני מהעבודה"
2. הסוכן שואל: "מה שמך המלא?"
3. הוא עונה: "דני כהן"
4. אתה מקבל התראה בטלגרם או בוואטסאפ
5. ענה: "כן" כדי להוסיף לאנשי הקשר
```

### תרחיש 4: שאלות ומידע
```
ג'מיני, מה המתכון לפיצה?
ג'מיני, תגיד לאשתי שאני בדרך הביתה
ג'מיני, מה המשמעות של חלום על נחשים?
```

## ⚠️ חשוב לזכור

### פורמט הפקודות
- **ג'מיני** - תמיד מתחיל במילה "ג'מיני" או "גמיני"
- **רשימות קניות** - מתחיל ב"רשימת קניות"
- **מחיקה** - מתחיל ב"מחק"
- **אישורים** - פשוט "כן" או "לא"

### שמות רשימות
- השתמש בסוגריים מרובעים: `[שם הרשימה]`
- אם לא מציין שם, ישתמש ב"רשימת קניות"

### חיפוש חכם
- למחיקת פריטים: מספיק לכתוב חלק מהשם
- לאנשי קשר: מספיק כינוי כמו "אשתי" או "אבא"

### התראות
- הודעות מאנשים לא מוכרים מגיעות לטלגרם (אם מוגדר)
- אישורי אירועים מגיעים לטלגרם או לוואטסאפ

- בדוק שהאפליקציה פועלת (השרת על port 3000)

- בדוק שהגדרת טלגרם נכון
- וודא שהבוט הוזמן לצ'אט

### אם רשימות לא נשמרות
- בדוק שיש Todoist API key
- וודא שהרשימה קיימת לפני מחיקה
