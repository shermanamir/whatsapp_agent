טוען סטטוס מערכת...

1️⃣ הגדרת משתני סביבה בודק...

צור קובץ .env בתיקיית הפרויקט עם המשתנים הבאים:

משתנה תיאור חובה סטטוס
GEMINI_API_KEY מפתח API מגוגל ג'מיני ✅ בודק...
TELEGRAM_BOT_TOKEN טוקן בוט טלגרם ❌ בודק...
TELEGRAM_CHAT_ID מזהה צ'אט טלגרם ❌ בודק...
TODOIST_API_KEY מפתח API טודואיסט ❌ בודק...

איך להשיג את המפתחות:

  • Gemini API: Google AI Studio
  • Telegram Bot: @BotFather בטלגרם
  • Todoist API: Todoist Settings

2️⃣ קבצי אישורים בודק...

צור קובץ credentials.json בתיקיית הפרויקט:

{ "installed": { "client_id": "YOUR_CLIENT_ID", "client_secret": "YOUR_CLIENT_SECRET", "redirect_uris": ["http://localhost"] } }

3️⃣ פריסה לענן בודק...

בחר ענן לפריסה:

🚂 Railway (מומלץ)

  1. הירשם ל-Railway
  2. חבר את GitHub
  3. דחוף את הקוד ל-repository
  4. Railway יזהה את הפרויקט ויפרוס אוטומטית

☁️ Google Cloud Run

  1. הירשם ל-Google Cloud
  2. צור Dockerfile
  3. השתמש ב-Cloud Build ו-Cloud Run

🔧 כללי לכל ענן:

  • הגדר את משתני הסביבה בענן
  • ודא שיש גישה ל-WhatsApp Web
  • שמור backup של תיקיית auth_info_baileys

4️⃣ התחברות לוואטסאפ בודק...

כדי להתחבר לוואטסאפ:

  1. הפעל את האפליקציה
  2. פתח http://localhost:3000 (או כתובת הענן)
  3. סרוק את קוד ה-QR עם אפליקציית וואטסאפ
  4. המתן לאישור החיבור

חשוב: שמור את תיקיית auth_info_baileys - היא מכילה את פרטי ההתחברות.

5️⃣ שימוש באפליקציה

אחרי ההתחברות, האפליקציה תענה אוטומטית להודעות:

פקודות זמינות:

  • ג'מיני [טקסט]: שאל שאלות מג'מיני AI
  • רשימת קניות: פריט1, פריט2: צור או הוסף לרשימת קניות ב-Todoist
  • רשימת קניות [שם הרשימה]: פריט1, פריט2: צור או הוסף לרשימה עם שם ספציפי
  • מחק רשימת קניות: מחק את רשימת הקניות הראשית
  • מחק רשימת קניות [שם]: מחק רשימה ספציפית
  • מחק את כל רשימות הקניות: מחק את כל רשימות הקניות
  • מחק מרשימת קניות: פריט: מחק פריט ספציפי מרשימה
  • הודעות מאנשים לא מוכרים: האפליקציה תבקש אישור דרך טלגרם
  • זיהוי אירועים: האפליקציה תזהה פגישות ותציע להוסיף ליומן

הגדרות טלגרם (אופציונלי):

הגדר טלגרם לקבלת התראות על הודעות חדשות ואישורים.

🔧 פתרון בעיות

בעיות נפוצות:

  • האפליקציה לא מתחילה: בדוק שכל משתני הסביבה מוגדרים
  • שגיאת חיבור לוואטסאפ: נסה למחוק auth_info_baileys וסרוק QR שוב
  • ג'מיני לא עובד: בדוק את ה-API key
  • טלגרם לא שולח: בדוק את ה-bot token וה-chat ID
